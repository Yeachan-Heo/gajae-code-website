Troubleshooting

Common Gajae Code setup, session, bridge, and CI problems with practical recovery checks.

Install and runtime

  • Confirm Bun and Node are on PATH.
  • Run gjc --help to confirm the CLI entrypoint resolves.
  • If native helpers fail, rebuild from a clean checkout and inspect platform-specific build output.

Session control

  • If a session stops accepting prompts, verify owner liveness before submitting again.
  • If branch metadata looks wrong, stop and correct the checkout instead of continuing in the wrong workspace.
  • Retire stale sessions after preserving their terminal outcome.

Bridge/RPC

  • Check endpoint URL, TLS/bearer configuration, and command scopes.
  • Treat denied scopes as a safety feature, not as a reason to widen authority blindly.
  • Prefer structured observations over log scraping.
